## Qu'est que le EvalExpr ?

Le EvalExpr est un sous-projet du Bistromatic.
Ce calculateur permet de faire des calculs avec des nombres négatifs et il gère parfaitement les prioritées des opérateurs.

### Compétences travaillées

- Premier projet de parsing
- Utilisation de la récursive
- Compréhension des opérateurs et des priorités

### Installation

Pour compiler ce projet, il vous suffit de vous rendre a la racine et de lancer la commande :

```bash
make
```

## Utilisation

Une fois le binaire généré, il vous suffit de le lancer grace a la commande suivante :

```bash
./eval_expr [expression]
```
